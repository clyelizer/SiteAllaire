Politique de confidentialité

Le Lycée Michel Allaire s’engage à protéger la vie privée des utilisateurs de son site web. Cette politique de confidentialité explique comment nous collectons, utilisons, stockons et protégeons vos données personnelles conformément aux réglementations applicables, notamment le Règlement Général sur la Protection des Données (RGPD) pour les résidents de l’Union européenne.

1. Collecte des données personnelles

Nous collectons les données personnelles suivantes lorsque vous interagissez avec notre site :

  • Formulaire de contact : Nom, adresse e-mail et contenu du message que vous soumettez via le formulaire de contact.
  • Espace étudiant : Informations de connexion (identifiant, mot de passe) pour accéder à l’espace numérique scolaire.
  • Données de navigation : Adresse IP, type de navigateur, pages visitées, et autres données collectées via des cookies ou technologies similaires (si applicable).

2. Finalités du traitement des données

Les données collectées sont utilisées pour les finalités suivantes :

  • Répondre à vos demandes envoyées via le formulaire de contact.
  • Fournir un accès sécurisé à l’espace étudiant et aux ressources pédagogiques.
  • Améliorer l’expérience utilisateur sur notre site (par exemple, via l’analyse des données de navigation).
  • Communiquer des informations sur les actualités, événements ou formations du lycée.

3. Base légale du traitement

Le traitement de vos données repose sur les bases légales suivantes :

  • Votre consentement (par exemple, pour les cookies ou l’envoi de messages via le formulaire).
  • L’exécution d’un contrat ou d’une obligation légale (par exemple, pour la gestion des comptes étudiants).
  • L’intérêt légitime du lycée à améliorer ses services et à communiquer avec ses utilisateurs.

4. Partage des données

Nous ne vendons ni ne louons vos données personnelles à des tiers. Vos données peuvent être partagées avec :

  • Les membres du personnel autorisés du Lycée Michel Allaire pour répondre à vos demandes.
  • Des prestataires de services techniques (par exemple, hébergeurs web ou services d’analyse) qui respectent les mêmes normes de confidentialité.

5. Sécurité des données

Nous mettons en œuvre des mesures techniques et organisationnelles pour protéger vos données personnelles contre tout accès non autorisé, perte ou altération. Cela inclut l’utilisation de protocoles sécurisés (HTTPS) et le chiffrement des données sensibles.

6. Durée de conservation

Vos données sont conservées uniquement pour la durée nécessaire à la réalisation des finalités décrites. Par exemple :

  • Les données du formulaire de contact sont conservées pendant [précisez une durée, par exemple, 1 an].
  • Les données des comptes étudiants sont supprimées à la fin de la scolarité, sauf obligation légale contraire.

7. Vos droits

Conformément aux lois applicables, vous disposez des droits suivants concernant vos données personnelles :

  • Droit d’accès : Consulter les données que nous détenons sur vous.
  • Droit de rectification : Demander la correction de données inexactes.
  • Droit de suppression : Demander la suppression de vos données, sous réserve des obligations légales.
  • Droit d’opposition : Vous opposer au traitement de vos données pour certaines finalités.
  • Droit à la portabilité : Recevoir vos données dans un format structuré.

Pour exercer ces droits, veuillez nous contacter via le formulaire de contact ou à l’adresse suivante : [insérez une adresse e-mail ou postale, par exemple, contact@lyceemichelallaire.org].
